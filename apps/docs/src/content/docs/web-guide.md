# Web Console Guide

The web console runs at `localhost:5173`. It enables the same interface controls as the desktop app for managing virtual reader networks.

![Dashboard](../../assets/dashboard.png)

## 1. Simulated RFID Tags

Navigate to the **Tags** tab to control the active tag inventory pool:

![Tags View](../../assets/tags_view.png)

### Tag Fields:
* **EPC Header**: Unique tag ID.
* **TID**: 8-byte silicon ID.
* **Read Count**: Detections count.
* **RSSI**: Dynamic signal strength.
