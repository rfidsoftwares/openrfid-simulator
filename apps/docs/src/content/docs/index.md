## Key Highlights

* **Headless CLI & APIs**: Run headless simulation servers via Fastify REST API, WebSocket streams, and MQTT telemetry brokers.
* **Virtual RFID Antennas**: Tune transmission power (dBm), antenna gain (dBi), and RSSI baseline offsets in real-time.
* **Physical Hardware Emulation**: Auto-discovered by Hopeland MyReaderAPI SDKs via UDP broadcasts and TCP command connections.
