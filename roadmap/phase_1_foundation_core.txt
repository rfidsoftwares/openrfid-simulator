2. CORE UTILITY PACKAGES IMPLEMENTATION
--------------------------------------------------------------------------------
[x] 2.1 Package: @openrfid/core
    - Location: packages/core
    - Implementation of Event Bus (typed event emitter for all simulator events)
    - Dependency Injection container pattern
    - Centralized Configuration Manager (schema validation, defaults)
    - Base Logger engine (structured JSON and pretty console output)
    - Storage Interface contracts
    - Component Lifecycle interfaces (Init, Start, Stop, Destroy)

[x] 2.2 Package: @openrfid/epc
    - Location: packages/epc
    - EPC Gen2 memory bank encoder and decoder
    - Support for hex, binary, and ASCII conversion utilities
    - Header validation and Bit-level manipulation tools
    - EPC checksum and filter value utilities

[x] 2.3 Package: @openrfid/gs1
    - Location: packages/gs1
    - SGTIN-96 (Serialised Global Trade Item Number) encoder/decoder
    - GRAI-96 (Global Returnable Asset Identifier) encoder/decoder
    - GIAI-96 (Global Individual Asset Identifier) encoder/decoder
    - Company Prefix and Item Reference translation tables

[x] 2.4 Package: @openrfid/utils
    - Location: packages/utils
    - Shared random number generators (Gaussian, uniform distributions for simulation)
    - Data format converters (CSV, JSON, XML parsers)
    - Network address and port validation utilities
