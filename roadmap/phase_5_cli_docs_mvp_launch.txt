1. CLI DEVELOPMENT
--------------------------------------------------------------------------------
[x] 1.1 Package: @openrfid/cli
    - Location: packages/cli
    - Command parser using Commander.js / Yargs
    - Commands implementation:
        * openrfid init -> Initializes a new simulator workspace / config file
        * openrfid simulator -> Starts standalone headless simulator server
        * openrfid create-reader -> Interactively creates a new virtual reader
        * openrfid create-tag -> Interactively generates virtual tags
        * openrfid mqtt -> Launches CLI MQTT monitoring session
        * openrfid export -> Exports reader/tag/event configurations
        * openrfid plugins -> Lists, enables, or installs plugins
    - Pretty terminal output with chalk, ora spinners, and cli-table3
